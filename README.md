# Enterprise AI Research Assistant

An enterprise-grade Retrieval-Augmented Generation (RAG) application built using LangChain, LangGraph, FastAPI, ChromaDB, Hugging Face Embeddings, and Google Gemini.

## 🚀 Features (Planned)

- PDF document ingestion
- Semantic search using vector embeddings
- Retrieval-Augmented Generation (RAG)
- AI Agent with LangGraph
- FastAPI backend
- Streamlit frontend
- Conversation memory
- Source citations
- Docker support

## 🛠️ Tech Stack

- Python
- FastAPI
- LangChain
- LangGraph
- ChromaDB
- Hugging Face Embeddings
- Google Gemini
- Streamlit
- Docker
